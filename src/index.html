Note: everything is in the gulpfile.js

Nothing fancy here. Just practice with gulp automation and a boilerplate. Moves the index file as-is. Places the css file from sass in the right place, and concatenates all js files (in the js folder)

Note any newer javascript used will create an error with no logging. Babel must be used before uglify is ran. And the errors will not pipe. A module called --pump-- must be added if you want to see errors from uglify!!!!

the command --gulp--, which is set as a default, runs all tasks from the command line in.
